# Changelog

## 0.1

### Added
- Command-Line Shell
- Basic Commands (`exit`, `help`, `clear`, `info`)
- Unreal Mode (16-bit)

## 0.2

### Added
- File Manager
- Built-in BASIC Scripting Language (BScript)
- Support for `.txt` files

### Changed
- Unreal Mode (32-bit) → Protected Mode (32-bit)

### Improved
- Better User Interface

## 0.3

### Added
- FAT12 File System
- Date & Time with Real Time Clock (RTC)
- User Login & Create Account
- Support for 24-bit BMP Images

### Improved
- Better File Manager

## 0.4

### Added
- PS/2 Mouse Cursor support
- Support for 32-bit BMP/JPG Images

### Changed
- Renamed Hardware Information labels:
CPU → Processor
RAM → Memory
Added Graphics information
- MIT License → GPL v2.

### Improved
- Better Hardware Info
- Better keyboard and mouse handling
- Improved Images Viewer

## 0.5

### Added
- USB Mouse and Keyboard support
- can't copy, move file and delete folder(S)
- FAT16 & FAT32 File System Support

### Improved
- Better keyboard and mouse handling
- Better File Manager
