# Contributing to FalconOS

Thank you for contributing to FalconOS!

Before submitting a Pull Request:

- Test your code.
- Follow the coding style.
- Explain what your changes do.
- Be respectful to other contributors.

If your contribution still contains issues, don't worry.
The maintainers will review it and help you improve it.

"Everyone is welcome to contribute. We review together, learn together, and build FalconOS together."
