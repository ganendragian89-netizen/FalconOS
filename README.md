### FalconOS

FalconOS is a lightweight, simple, free, and open-source **command-line operating system** built from scratch using x86 Assembly and C.

## Minimum Specifications

- CPU      : x86 Processor (80486+)
- RAM      : 4 MB
- Storage  : 32 MB Free Space
- Firmware : Legacy BIOS
- Display  : VGA Compatible
- Output   : PS/2 Keyboard, Mouse
- RTC      : Required

## Recommended Specifications

- CPU      : Pentium or newer
- RAM      : 16 MB+
- Storage  : 128 MB Free Space
- Firmware : Legacy BIOS
- Display  : VGA/SVGA
- Output   : USB Keyboard, Muse
- RTC      : Required

## Features

- Command-Line Shell
- Built-in BASIC Scripting Language (BScript)
- Built-in Text Editor
- File Manager
- Date & Time Management (RTC)
- User Account
- BMP Image Viewer (24-bit BMP)
- Multi-language Interface **(Coming Soon)**
  - English Language (Default)
  - Bahasa Indonesia
  - العربية
  - 中文
- Lightweight
- Fast
- Simple
- Open Source
- Built from Scratch

## Design Philosophy

FalconOS is designed to be:

- Hobby project Operating System
- Lightweight and easy to understand.
- Beginner-friendly for learning operating systems.
- Built with user freedom while respecting user privacy.
- Expandable through built-in scripting.
